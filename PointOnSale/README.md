# PointOnSale

## Prerequisites

- .NET SDK (for backend)
- Node.js (for mobile)
- Expo CLI (for mobile)

## Backend

1. Open a terminal and move into the backend folder:
   ```bash
   cd PointOnSale/backend
   ```
2. Restore dependencies (once you have a .NET project here):
   ```bash
   dotnet restore
   ```
3. Run the backend (once you have a .NET project here):
   ```bash
   dotnet run
   ```

## Mobile

1. Open a terminal and move into the mobile folder:
   ```bash
   cd PointOnSale/mobile
   ```
2. Install dependencies (once you have a Node/Expo project here):
   ```bash
   npm install
   ```
3. Start the Expo dev server:
   ```bash
   npx expo start
   ```

## Environment files

- Backend env files: `PointOnSale/backend/.env` (or project-specific configuration files).
- Mobile env files: `PointOnSale/mobile/.env` (or Expo config files as needed).
